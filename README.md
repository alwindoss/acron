# acron

### Acronym Store

## Install
 `go get -u github.com/alwindoss/acron`

 Add `$GOPATH/bin` to `$PATH`

## Commands
 `acron add --key SRS --value "Software Requirements Specification"`
 
 `acron get --key SRS`
